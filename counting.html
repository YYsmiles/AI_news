Quick choice

怎么选

没有标注数据

优先试 CountGD / CountGD++：输入图片和文字提示，输出数量和位置。

有点标注数据

训练 P2PNet / PET / LOCA 这类专用计数模型，稳定性通常更好。

需要可解释结果

不要只返回数字，最好展示 boxes / points / heatmap，方便人工审查。

路线 模型 适合 指标 入口
开放世界计数 CountGD / CountGD++Oxford VGG 类别经常变化、标注少、需要文字提示。 MAE、RMSE、Bias、定位误差。 Project
固定场景计数 P2PNet / PET / LOCAResearch baselines 同一类目标长期重复计数，例如花朵、细胞、人群。 MAE、RMSE、Bias、跨地块泛化。 GitHub
检测后计数 YOLO / GroundingDINODetection pipeline 目标清晰、遮挡少、需要 boxes 而不是点。 Precision、Recall、F1、count error。 Docs
